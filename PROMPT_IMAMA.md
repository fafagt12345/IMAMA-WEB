# Prompt Tambahan – Website IMAMA

## 9. Penambahan Menu Event pada Navbar Website
Pada navbar website publik saat ini terdapat menu:

- Beranda
- Tentang
- Struktur
- Program
- Galeri
- Kontak
- Login

Tambahkan menu baru:

### Event
Posisi menu:

Beranda | Tentang | Struktur | Program | Event | Galeri | Kontak | Login

Fitur halaman Event:

- Menampilkan seluruh event yang sudah dibuat dari dashboard admin.
- Menampilkan banner event.
- Menampilkan tanggal event.
- Menampilkan lokasi event.
- Menampilkan deskripsi singkat.
- Tombol "Lihat Detail".
- Event terbaru tampil paling atas.
- Bisa difilter berdasarkan:
  - Tahun
  - Bulan
  - Kategori

Tambahkan fitur pencarian event.

## 10. Menu Titik Tiga (Hamburger Menu)
Tambahkan tombol menu titik tiga atau hamburger menu pada navbar.

### Tampilan Desktop
Saat tombol ditekan:

- Navbar bergeser secara smooth.
- Muncul panel menu tambahan dari samping kanan.
- Animasi slide.
- Tidak merusak layout yang sudah ada.

Menu yang muncul:

- Beranda
- Tentang
- Struktur
- Program
- Event
- Galeri
- Kontak
- Login

### Tampilan Mobile
Saat tombol ditekan:

- Menu dropdown muncul dari atas ke bawah.
- Full width.
- Animasi smooth.
- Mudah digunakan dengan satu tangan.

## 11. Navbar Modern
Perbaiki navbar agar lebih modern:

- Tambahkan tombol hamburger/titik tiga di sebelah kanan.
- Navbar sticky saat scroll.
- Efek blur transparan saat scroll.
- Animasi hover pada menu.
- Active menu indicator.
- Responsive di semua perangkat.

## 12. Integrasi dengan Dashboard Admin
Menu Event pada website publik harus otomatis mengambil data dari menu Event & Lomba yang ada di Dashboard Admin.

Jika admin:

- Menambah event → otomatis muncul di halaman Event.
- Mengedit event → otomatis terupdate.
- Menghapus event → otomatis hilang dari website.

Tidak perlu mengubah kode secara manual.

Catatan Penting:
Jangan menghapus desain navbar yang sudah ada. Hanya tambahkan menu Event dan tombol titik tiga/hamburger dengan animasi modern, responsif, dan tetap konsisten dengan tema hijau IMAMA UNESA yang saat ini digunakan.
